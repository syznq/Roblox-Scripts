## How to Use

### Step 1
Get a script executor (like Volcano, Bunni, Xeno, Solara or others)

### Step 2
Join a Roblox game that's supported

### Step 3
Copy the script you want from this repository

### Step 4
Paste it into your executor and run it

### Load Script Method
```lua
loadstring(game:HttpGet("RAW_SCRIPT_URL_HERE"))()
```
